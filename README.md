Проект посвящён классификации символов алфавита американского жестового языка (ASL) по изображению. На данный момент рассматривается инженерная постановка задачи и создаётся базовая структура проекта для реализации baseline.

Для проекта используется датасет ASL Alphabet (https://www.kaggle.com/datasets/grassknoted/asl-alphabet).

В данный момент baseline ещё не реализован.
